# Demo Screenshots and Assets

This folder contains screenshots, GIFs, and other visual assets for the README.

## Recommended Screenshots

1. **main_interface.png**: Overview of the app homepage
2. **single_prediction.png**: Example of single molecule prediction with heatmap
3. **batch_analysis.png**: Batch processing interface
4. **toxic_example.png**: Explainability heatmap for a toxic molecule
5. **safe_example.png**: Explainability heatmap for a safe molecule

## Creating Demo GIFs

Use tools like:
- **Kap** (macOS): https://getkap.co/
- **ScreenToGif** (Windows): https://www.screentogif.com/
- **Peek** (Linux): https://github.com/phw/peek

Recommended demo flow:
1. Enter SMILES → view structure → check Lipinski
2. Get prediction → analyze heatmap → download PNG
3. Upload batch CSV → get results → download
